Getting Stuff from PDFs

  • Document Generation

Adobe PDF Extract

  • Uses Adobe Sensei (ML, AI, Skynet, etc)
  • Extracts text, tables, images, styling information
  • Tables can be CSV, XLSX, or images
  • Extracts document structure
  • Auto OCRs when necessary
